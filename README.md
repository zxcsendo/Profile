# Joshua Trajano

A stunning professional portfolio website featuring 3D graphics, smooth animations, and a bold red & black gradient design.

## Tech Stack

- **React** + **TypeScript** + **Vite**
- **Three.js** / **React Three Fiber** — 3D hero scene
- **Framer Motion** — scroll & entrance animations
- **Lucide React** — icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```

## Sections

- **Hero** — 3D floating orb with particle field
- **About** — personal introduction
- **Services** — expertise cards
- **Work** — portfolio projects
- **Skills** — animated progress bars & tools
- **Contact** — form & info
